# OpenDXP Element Manager Documentation

The OpenDXP Element Manager bundle adds configurable save handling for data objects and a duplicate checker based on Symfony validation constraints.

## Contents

- [Installation](installation.md)
- [Configuration](configuration.md)
- [Save Manager and Lifecycle](save-manager.md)
- [Naming Schemes](naming-schemes.md)
- [Duplicate Checker](duplication-checker.md)
- [Validation on Save](validation.md)
- [Custom Save Handlers](custom-save-handlers.md)

## At a glance

- Register per-class save managers that react to pre/post add/update/delete events.
- Enable naming schemes, unique keys, validation, and duplicate checks per class.
- Define duplicate rules with YAML mapping files and validation groups.
- Extend behavior with your own save handlers.
